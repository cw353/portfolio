Technical Editor & Contributing Author for Interactive Mathematics Textbook (Summer 2023 – Present)

This open-source, online textbook seeks to provide math educational resources to under-resourced communities. It focuses on interactivity and the integration of Python programming using Jupyter Notebooks. The textbook is being developed by a team of faculty members and student researchers from Wheaton College and other educational institutions, including my faculty mentor Dr. Peter Jantsch.

  • Online version of textbook
    • Section on Statistical Inference to which I was a contributing author (including work on both interactive and non-interactive graphics)
  • Repository containing source code
  • Primary languages, libraries, & tools: Python, Jupyter Books, Jupyter Notebooks

Cook County Sales Map (Summer 2023)

This interactive map facilitates both temporal and spatial analysis of patterns in property sales in Cook County, IL. It was developed to assist Chicagoland nonprofits with research and data analysis aimed at combating housing inequity.

  • Repository containing source code
  • Primary languages, libraries, & tools: JavaScript, JQuery, Leaflet, HTML, CSS
