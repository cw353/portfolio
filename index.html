Technical Editor & Contributing Author for Interactive Mathematics Textbook (Summer 2023 – Present)

This open-source, online textbook seeks to provide math educational resources to under-resourced communities. It focuses on interactivity and the integration of Python programming using Jupyter Notebooks. The textbook is being developed by a team of faculty members and student researchers from Wheaton College and other educational institutions, including my faculty mentors Dr. Peter Jantsch, Dr. Paul Isihara, and Dr. Ying Li.

  • Online version of textbook
    • Section on Statistical Inference to which I am a contributing author
    • Section on Advanced Data Analysis with Natural Language Processing to which I am a contributing editor
  • Repository containing source code
  • Primary languages, libraries, & tools: Python, Jupyter Books & Notebooks, LaTeX, SciPy, statsmodels, scikit-learn, NLTK, spaCy, fastText, Hugging Face, Googletrans, Matplotlib, Plotly, NumPy, pandas

Researcher on REMICSS Protocol (Spring 2023 – Fall 2023)

I worked with Dr. Devin Pohly on REMICSS, a network-layer multi-channel secret sharing protocol, as part of a team of student researchers. My team and I assisted with design, implementation, and experimentation. In particular, we worked on encapsulating the REMICSS protocol into a library patterned after the POSIX sockets API. I also served as a contributing author to the paper "Dynamic Security Parameters for Multichannel Secret Sharing Protocols" presented at ITCCN-2023.

  • Primary languages, libraries, & tools: C, glibc

Cook County Sales Map (Summer 2023)

This interactive map facilitates both temporal and spatial analysis of patterns in property sales in Cook County, IL. It maps trends in property sales across both time and space while providing graphical tools and statistical summaries for visualizing and analyzing the data. The map was developed to assist Chicagoland nonprofits with research and data analysis aimed at combating housing inequity.

  • Interactive map
  • Repository containing source code
  • Primary languages, libraries, & tools: JavaScript, JQuery, Leaflet, Plotly, HTML, CSS

HEI Map Project (Summer 2022)

This map provides an interactive visualization of geospatial data related to historic and contemporary housing inequity in Cook County, IL. It plots a wide variety of housing-related geospatial data ranging from historical redlining data to recent tax sale records, allowing for comparative analysis. The map was developed to assist Chicagoland nonprofits with research and data analysis aimed at combating housing inequity.

  • Interactive map

  • Primary languages, libraries, & tools: JavaScript, JQuery, Leaflet, HTML, CSS

Stylometry Project (Fall 2023)

Project for Computational Linguistics course. Using computational tools, I analyzed stylistic differences between early mystery novels by Dorothy L. Sayers and Agatha Christie. Specifically, I investigated how frequencies of types and parts of speech reflect each writer's unique stylome and can thus be used for authorship attribution.

  • Report in Jupyter Notebook format
  • Report in HTML format
  • Primary languages, libraries, & tools: Python, NLTK, scikit-learn, Matplotlib, seaborn, NumPy, pandas

Machine Learning Project (Spring 2023)

Term project for Machine Learning course. In collaboration with a classmate, I experimented with a variety of machine learning techniques, ranging from K Nearest Neighbors to Multilayer Perceptrons, with the goal of diagnosing early stage diabetes with maximal accuracy and sensitivity.

  • Report in Jupyter Notebook format
  • Report in HTML format
  • Primary languages, libraries, & tools: Python, scikit-learn, Matplotlib, NumPy, pandas

Mathematical Statistics Project (Spring 2023)

Term project for Mathematical Statistics course. In collaboration with two other classmates, I created a multiple regression model and performed a variety of statistical tests to analyze factors influencing duration of stay in animal shelters.

  • Report in R Markdown format

  • Primary languages, libraries, & tools: R, Tidyverse, LaTeX
